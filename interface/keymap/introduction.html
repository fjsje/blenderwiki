通用快捷键

本手册中使用的惯例

键盘

在本手册呈现的字符快捷键，如下所示：

G

等同于小写的 g。

Shift、Ctrl、Alt

特指修饰键。

Ctrl-W、Shift-Alt-A ...

表示应同时按下这些键。

数字键盘0 至 数字键盘9、数字键盘加+

表示小键盘上的独立键位。

其他键位如它们的名字一样 Esc、Tab、F1 至 F12。需要特别注意的是 左←、右→ 等。

鼠标

本手册鼠标术语预定如下：

鼠标左键

鼠标左键

鼠标右键

鼠标右键

鼠标中键

鼠标中键

滚轮、滚轮向上 与 滚轮向下

鼠标滚轮。

Note

Blender 有两种主要的选择模式：左键选择和右键选择。参见鼠标按键选择设置。

虽然左键选择是默认的，因为它在其他应用程序中是最常见的，但右键选择确实有其优势。见：了解右键选择的好处。

悬停

当悬停时（当鼠标放在一个输入框上）。

属性

  • Ctrl-C -- 复制输入框里的值（单个）。

  • Ctrl-V -- 粘贴值进输入框（单个）。

  • Ctrl-Alt-C -- 复制输入框里整个向量值或色彩值。

  • Ctrl-Alt-V -- 粘贴整个向量值或色彩值进输入框里。

  • 鼠标右键 -- 打开上下文菜单。

  • 回退键 -- 清除值（归零或清楚内容）。

  • 减号- -- 数值取负（乘以 -1.0）。

  • Ctrl-滚轮 -- 递增变化。

    对于弹出选项菜单按钮，将循环选择。

  • 回车 -- 选择菜单项或切换值。

  • Alt -- 在编辑时按住，会把操作应用到所有选择项中。（如物体、骨骼、视频片段）。

    这对于数值栏或开关操作非常有用（译者注：如同时更改灯光亮度时，你可以同时选择几个灯光，然后按住alt键，同时调节灯光亮度）。

动画

  • I -- 插入关键帧。

  • Alt-I -- 清除关键帧。

  • Shift-Alt-I -- 清除所有关键帧（删除所有样条曲线）。

  • Ctrl-D -- 添加驱动。

  • Ctrl-Alt-D -- 清除驱动。

  • K -- 添加插帧集。

  • Alt-K -- 清除插帧集。

Python 脚本

  • Ctrl-C 在任何操作按钮上复制其 Python 命令到剪贴板。

    这在 Python 控制台或者在文本编辑器中写脚本时非常有用。

  • Shift-Ctrl-C 在属性按钮上复制其该属性的数据路径（也可从上下文菜单中获得）。

    对写驱动脚本时非常有用。

  • Shift-Ctrl-Alt-C 在属性按钮上复制其 完整 的数据块和属性的数据路径。

    需要注意在大多数情况下，最好根据正文而不是名称来访问值。

拖拽

  • Ctrl -- 拖动时吸附倍数步长。

  • Shift -- 精确控制拖拽的值。

  • Shift-Ctrl -- 伴随高精度的移动物体时的精确捕捉约束。

文本编辑

  • Home -- 转到该行的开头。

  • End -- 转到该行的结尾。

  • 左←, 右→ -- 把光标移动一个字符距离。

  • Ctrl-左←, Ctrl-右→ -- 把光标移动一个单词距离。

  • 回退键, Delete -- 删除字符。

  • Ctrl-回退键, Ctrl-Delete -- 删除文字。

  • Shift -- 按住键并移动光标选择。

  • Ctrl-A -- 选择所有文本。

  • Ctrl-C -- 复制选择的文本。

  • Ctrl-X -- 剪切选择的文本。

  • Ctrl-V -- 在光标的位置粘贴文本。

确认和取消

  • Esc, 鼠标右键 -- 取消。

  • 回车, 鼠标左键 -- 确认。
