随机值

随机值 节点输出一个类似噪点的值，作为一个 浮点、整数、矢量 或 布尔 字段。

输入

最小值

抽取随机值的范围的最小值。这个输入仅适用于 浮点、整数 和 矢量 类型。

最大值

抽取随机值的范围的最大值。这个输入仅适用于 浮点、整数 和 矢量 类型。

概率

随机 布尔 输出为 真 的概率比。这个输入仅适用于 布尔 类型。

ID

用于驱动随机数发生器种子的 ID。默认情况下，这个输入使用与 ID 相同的值，如果存在的话，就是上下文几何的 id 属性，否则就是编号。

Tip

单个随机值

默认情况下，随机值节点为每个唯一的编号生成一个值。如果需要一个单一的随机值，请将一个单一的值（例如整数）连接到 ID 输入。

随机种

一个字段，用于 种子 随机数发生器。这可以用来生成一组不同的随机值，甚至对两个具有相同 ID 输入的节点。

属性

数据类型
浮点:

输出将会是一个 浮点 场。

整数:

输出将会是一个 整数 场。

矢量:

输出将会是一个 矢量 场。

布尔:

输出将会是一个 布尔 场。

输出

数值

某一字段的随机值。
