预览器节点

预览器 节点允许从电子表格编辑器和 3D 视口中的几何节点组内查看数据。

任何连接的几何体都可以在视口中可视化，其属性值也可以在电子表格中读取。

用法

激活与停用

在任何节点或接口上使用 Shift-Ctrl-鼠标左键 都会将其连接到预览器并使其处于激活状态。在节点编辑器中的空白处点击相同的快捷键可以停用活动预览器。当活动预览器不再可见时（例如选择了另一个物体或退出了当前节点组），它就会被停用。预览器节点标题中的图标也可用于激活或停用预览器。

在视口中，显示预览器 选项可以完全关闭预览器节点可视化，以查看物体评估的最终输出结果。

属性可视化

When the viewer has a geometry and a separate value input connected, the values can be visualized with a viewport overlays. When possible, the attribute domain used to visualize the data is determined automatically. Otherwise, the viewer node falls back to the face corner domain on meshes and the point domain on curves. When necessary, the domain can be chosen manually.

The attribute overlay opacity can be controlled with the Viewer Node setting in the overlays popover.

The spreadsheet now only shows the "Viewer" column for the domain that is selected in the Viewer node.

钉住

在电子表格中固定一个特定的预览器节点可能很有帮助。固定后，即使预览器节点处于非活动状态，电子表格仍会引用该节点。

输入

几何数据

将被显示在电子表格中的几何数据。

数值

要在几何体上进行评估的场。按下键盘快捷键连接输出时，将自动选择该值的类型。但是，如果必须手动调整类型，则可在节点编辑器侧栏中进行更改。

属性

数据类型

用于评估 值 输入项的数据类型，在节点侧栏中可见。

域

用于评估 值 输入项的属性域。自动 选项表示基于已连接的节点自动选择域。

输出

此节点没有输出。
