顶点倒角

参考

模式:

编辑模式

菜单:

边 ‣ 边线倒角

快捷键:

Ctrl-B (边倒角)

菜单:

顶点 ‣ 顶点倒角

快捷键:

Shift-Ctrl-B (顶点倒角)

倒角 工具在选定顶点的位置上将网格的边缘或角落磨圆。在 "(仅)顶点" 模式下，顶点倒角 工具在选定的顶点上工作，但也可以选择切换到 边线倒角。通过这样做，更多的顶点被添加进来，以便用指定数量的 段 来平滑剖面（关于倒角算法的细节，见下面的选项）。

Note

在激活 "仅顶点" 选项时，其他一些可用的选项将无法工作。但是，它们可以与边倒角一起使用。

选项

影响 V
顶点:

仅靠近顶点的区域会倒角，边不受倒角影响。

边:

倒角边，在顶点处创建交集。

偏移 A

你可以通过将鼠标移向或移开物体来改变倒角的数量，就像使用变换工具一样。该值的确切含义取决于 宽度类型 选项（见下文）。

宽度类型 M

用于选择 数量 对倒角程度的计算方式，对应的类型有：

偏移量:

新生边到原位之间的相对距离。

宽度:

倒角形成的两条新边的距离（如果有多个段，则为坡口两侧的边）。

深度:

原始边到倒角面的垂直距离。

百分比:

新边线沿相邻边长度滑移的百分比。

绝对:

沿着与倒角边相邻的边的确切距离。当连接到倒角边的非倒角边与直角以外的角度相遇时，可以看到与 偏移量 的差异。

对于纯顶点倒角，偏移量和深度类型从原始顶点开始计量，且宽度类型以新顶点到新面的中心计量（作为 宽度 的一半）。

段数 S

滚动 滚轮 增加或减少倒角面段数。段数越多，倒角越平滑。也可以按下 S，移动鼠标或输入数值来改变段数。

另外，在使用该工具时或结束操作后的操作面板上，可以手动输入段数值。

轮廓 P

大小介于 0 到 1 之间，用于控制倒角的轮廓（倒角边侧视）形状。默认值为 0.5，轮廓为圆弧形（如果这些面以正确的角度相连）。低于该值，轮廓变得平坦，0.25 为平面，低于 0.25 为内凹的倒角。该值大于 0.5，倒角面更加突出。与 段数 类似，该值可以在按下 P 后移动鼠标或输入数值修改。

材质编号

材质 编号用于指定赋予新 倒角 面的材质，默认为 -1，继承最近的现有面的材质（"最近" 似乎有些含糊）。否则，将为新的倒角面赋予材质槽中对应编号的材质。

硬化法向 H

When enabled, the per-vertex face normals of the bevel faces are adjusted to match the surrounding faces, and the normals of the surrounding faces are not affected. This will keep the surrounding faces flat (if they were before), with the bevel faces shading smoothly into them. For this effect to work, custom split normals need to be enabled, which requires Auto Smooth to be enabled (see Normals). As a convenience, that option will be enabled for you if it is not already when you enable Harden Normals here.

钳制重叠 C

限制每个倒角边的宽度，使顶点不与其他几何体重叠。

环切线滑移

如果一个顶点有未倒角的边和已倒角的边，倒角会尽可能沿着这些边滑动。关闭该选项可以产生更均匀的倒角宽度。

面强度模式

Set Face Strength on the faces involved in the bevel, according to the specified mode. This can be used in conjunction with a Weight Normals Modifier (with the Face Influence option checked).

无:

不设置面强度。

新建:

设置沿边缘的新面的面强度为 中，顶点的新面的面强度为 弱。

受影响的:

除了为设置为 新建 的那些面外，还将相邻的面设置为具有 强 强度。

全部:

除了那些设置为 受影响的 选项，还设置模型的所有其余面强度为 强。

轮廓类型 Z
超椭圆:

创建具有均匀凹面或凸面曲线的倒角。

自定义:

此部件用于创建用户定义的轮廓，其复杂性比使用单个轮廓参数更复杂。模态工具允许切换自定义轮廓，但仅在确认操作后，才能在选项面板中编辑轮廓形状。

轮廓从小部件的右下方开始，在左上方结束，就像在两条相交于直角的边缘之间。控制点在小部件中被创建，然后用倒角修改器的段数对路径进行采样。

预设

支撑环 和 阶梯 预设根据倒角中的段数动态生成。如果段数发生更改，则必须重新应用预设。

采样

样本将首先被添加到每个控制点上，然后如果有足够的样本，它们将被平均分配到边缘上。采样直边 选项可以切换是否将样本添加到两侧有尖锐控制点的边上。如果没有足够的样本来给每个边缘提供相同数量的样本，它们将只是被添加到最弯曲的边缘上。所以建议至少使用与控制点相同数量的分段数。

启用了自定义轮廓的倒角。
