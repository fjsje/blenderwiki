路径动画

路径动画 设置可用于确定子物体如何沿着特定路径移动。

Note

这个特性是不推荐的，但它仍然可用。一个更适合使用的方法是Follow Path Constraint 跟随路径约束。

帧

遍历路径所需的帧数，定义 估算时刻 设置的最大值。

估算时刻

沿曲线长度方向的参数位置，即追踪对象应处于的位置（该位置通过除以 帧 值进行评估）。默认情况下，它与全局帧数相关联，但也可以通过关键帧对路径动画进行更多控制。

钳制

钳制曲线路径的子物体，使他们不能超过曲线的 起始/结束 点。

跟随

使曲线路径的子物体沿路径的曲率旋转。

例子

这个例子向您展示了如何设置一个 路径动画。

  1. 添加一个要动画化的物体和该物体将沿其移动的路径。在这个例子中，它是 猴头 和 贝塞尔圆。

  2. 将该猴头的父级设置为贝塞尔圆，首先选择该猴头，然后选择曲线（以使该曲线是活动物体），按 Ctrl-P，然后选择 跟随路径。它将自动地动画化 估算时刻，并激活 路径动画 面板中的 跟随 选项。

  3. 选择猴头并清空原点以重置它的偏移量。

  4. 可以通过改变跟踪坐标轴来改变猴头的方向。
