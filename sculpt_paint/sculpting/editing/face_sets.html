面组

This page details the face set related hotkey operators and menu operators in sculpt mode.

从被遮蔽面创建面组

参考

模式:

雕刻模式

菜单:

面组 ‣ 从被遮罩面创建面组

从 几何遮罩 创建新的面集。

从可见面创建面组

参考

模式:

雕刻模式

菜单:

面组 ‣ 从可见面创建面组

从所有可见几何创建新面集。

从编辑模式选中项创建面组

参考

模式:

雕刻模式

菜单:

面组 ‣ 从编辑模式选择面集

创建与 "编辑模式" 面选择相对应的新面组。

初始化面组

参考

模式:

雕刻模式

菜单:

面组 ‣ 初始化面组

根据多个网格属性之一一次初始化网格上的所有面集。

模式

用于定义面集边界的网格数据属性。

按松散块

为网格的每个不连续部分创建新的面集。

按面组边界

Creates a face set for each isolated face set. This mode is useful for splitting the patterns created by Face Set Expand into individual Face Sets for further editing.

按材质

根据 材质槽 创建面集。

按正常值

为具有类似 法线 的面创建面组。

按UV缝合边

使用 UV接缝 作为边界创建面集。

按边折痕

使用 边缘折痕 作为边界来创建面集。

按边线倒角权重

使用倒角权重作为边界来创建面集。

按锐边

使用 锐边 作为边界来创建面集。

By Face Maps

Creates a face set per Face Map.

阈值

创建面集时将特定属性视为边界的最小值。

增大/缩小面集

参考

模式:

雕刻模式

菜单:

面组 ‣ 扩展/收缩面组

工具:

编辑面组

快捷键:

Ctrl-W, Ctrl-Alt-W

通过添加或删除周围的面来扩展或收缩光标下的面集。

扩展面组

Note

More info on Face Set Expand at the Expand page.

提取面组

参考

模式:

雕刻模式

菜单:

面组 ‣ 提取面组

基于所选面组创建新网格。启动操作后，将鼠标悬停在面组上并点击 鼠标左键 以创建新网格。操作完成后，新的网格将在物体模式下被选中。

Invert Visible Face Sets

参考

模式:

雕刻模式

菜单:

面组 ‣ 反转可见面组

Hides all geometry that is part of a face set and makes all hidden geometry that is part of a face set visible.

Show All Face Sets

参考

模式:

雕刻模式

菜单:

面组 ‣ 显示所有面组

快捷键:

Alt-H

Shows all hidden geometry that is part of a face set.

随机化颜色

参考

模式:

雕刻模式

菜单:

面组 ‣ 随机化颜色

生成一组新的随机颜色以在 3D 视口中呈现面组。

显示设置

参考

模式:

雕刻模式

弹出框:

视图叠加层 ‣ 雕刻 ‣ 面组

面组可作为 视图叠加层 切换显示。在叠加弹出框中，可以调整面组叠加的不透明度，使其在网格上提升或降低可见性。
