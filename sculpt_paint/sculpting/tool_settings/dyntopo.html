动态拓扑

参考

模式:

雕刻模式

面板:

侧栏 ‣ 工具 ‣ 动态拓扑

快捷键:

Ctrl-D

Dynamic Topology (aka Dyntopo) can be toggled with the checkbox in the header or with Ctrl-D. With dynamic topology active, most brushes will subdivide the mesh during the stroke.

For a general explanation of dynamic topology, visit the Introduction.

细节尺寸/百分比、分辨率 R

每个细节类型的详细信息都在此处设置。根据所使用的细节类型，此属性将显示为像素数（px）或百分比。

采样细节尺寸（滴管图标）

使用 常量细节 时，可以通过单击细节设置旁边的移液器图标，然后单击该区域来采样特定网格区域的细节值。

改进方法

设置该选项将确定在更改拓扑时将使用哪些方法。

细分边线:

就像细分工具一样，此方法只会细分拓扑以匹配给定的细节。

塌陷边线:

当拓扑过于密集且小于给定的细节时，边缘将塌陷以适当地适配细节大小。

细分塌陷:

此方法结合了两种方法，细分小于细节大小的边，以及塌陷拓扑。

细节

Dyntopo uses three different detail methods to create dynamic detail to an object.

相对细节:

此方法使用基于像素数的细节大小，然后将创建该大小的拓扑。视图缩小时细节偏大，视图放大时细节趋向小而精细。

恒定细节:

To keep detail uniform across the entire object, Constant Detail can be used. The Detail is based on the percentage of a single unit.

笔刷细节:

通过对拓扑进行更多控制，可以使用此方法根据画笔大小创建拓扑。您可以通过调整画笔本身的大小来增加和降低拓扑。细节大小基于画笔本身的大小，其中全部细节将创建画笔半径本身大小的拓扑。

细节批量填充

使用 常量 或 手动 细节 时，此选项可用，允许您根据细节大小用统一的细节填充整个对象。

平滑着色

Toggles whether mesh faces are smooth or flat-shaded. In dynamic-topology mode all faces have the same type of shading.
