在 macOS 上安装

在 下载 Blender 页面确认最低配置要求以及 Blender 的不同版本（如果您还没这么做的话）。

从 DMG 安装

用于 macOS 的 Blender 分发在磁盘映像（dmg 文件）上。要挂载该磁盘映像，请双击 dmg 文件。然后将 Blender.app 拖到应用程序（Applications）文件夹中。

根据您的 Mac 的 "安全性和隐私" 偏好设置，在首次打开 Blender 之前，macOS 将要求您的批准。

Tip

如何实现便携安装

要将所有配置文件和已安装的插件保存在 Blender 程序包中，请在 LOCAL 目录 中创建一个名为 config 的文件夹。

./Blender.app/Contents/Resources/3.6/config/

在 macOS 上更新

在 macOS 上，有多种更新 Blender 的方法。本节介绍最常用的方法。

使用 DMG 更新

当 Blender 的更新发布后，可以直接从 Blender 网站 下载。通过覆盖应用程序（Applications）文件夹中的 Blender.app 的当前版本来安装新版本。您可以将 Blender.app 重命名或放置在一个不同文件夹中，以同时拥有多个版本。

See also

启动画面默认页，提供有关从以前的 Blender 版本导入配置以及其他快速设置的信息。
