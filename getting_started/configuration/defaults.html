默认

当您首次启动 Blender，或升级到新版后，在启动画面的互动区域，被替代为了一组初始的偏好设置，以配置 Blender 的交互方式。

Note

这些设置也可以之后在偏好设置进行修改。

语言

用户界面使用的语言。该列表根据翻译的完成度分为几类。更多语言相关设置见翻译设置。

快捷键

默认 Blender 键位映射。请注意，本手册假定您使用 “Blender” 键位映射。

Blender:

默认的键位映射。点击这里了解更多。

Blender 2.7x:

这种键位映射是为了兼容近期的 Blender 重要版本，适用于更新版本后不愿意使用最新版键位映射的人群。

行业兼容:

这种键位映射兼容常见的同类软件，适用于使用不同软件的人群。点击这里了解更多。

选择方式

配置通过鼠标左键或鼠标右键在 Blender 中选择物体。

空格键

配置 空格键 的对应动作。这些和其他快捷键可通过键位映射设置修改。

播放:

开始播放时间线。这个选项适用于动画或视频剪辑。

工具:

在鼠标箭头下方打开工具栏来快速切换工具。这个选项适用于建模或绑定。

搜索:

打开搜索菜单。这个选项适用于对 Blender 菜单和快捷键不熟悉的新手。

主题

选择 Blender 的亮色或暗色主题。可以在主题设置里配置个性化主题。

加载以前版本的设置

从先前版本的 Blender 复制偏好设置和启动文件，然后加载它们。

由于每个 Blender 版本的配置文件存储在单独的文件夹中，因此需要从以前的版本中导入设置。有关这些文件夹的位置，请参考 Blender 目录布局页面。

Blender 的默认配置包括两部分：

偏好设置

偏好设置文件保存了键位映射、插件主题和其他选项。

启动文件

启动文件保存了新建文件时后默认打开的场景和界面设置（文件 ‣ 新建）。

保存默认设置

有变化时偏好设置会自动保存。

通过 文件 ‣ 默认 ‣ 保存启动文件 可以修改默认启动文件。参见启动文件。

加载初始设置

你可以恢复自定义设置为 Blender 默认设置：

偏好设置

偏好设置可以恢复初始设置。

启动文件和偏好设置

文件 ‣ 默认 ‣ 加载初始设置。

Note

恢复初始设置后，设置不会自动保存。

详情见管理偏好设置。
