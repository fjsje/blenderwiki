Blender 社区

即使在闭源阶段，Blender 也是可以免费获取的，这很大程度上帮助 Blender 获得了社区的接纳。自 1998 年起，一个大型、稳定并且活跃的用户社区已经围绕 Blender 聚集起来。2002 年，社区展示了其支持的力量：在七周内筹集了十万欧元，帮助 Blender 转变为 GNU GPL 许可证下发行的开源软件。

独立站点

有几个独立的网站如致力于 Blender 论坛、博客、新闻和教程网站。

Blender Artists 是最大的社区论坛之一，Blender 用户聚集在这里展示他们的作品，获得反馈，提问和提供帮助，并且一般都是讨论 Blender。

获得支持

Blender 社区是其最大的特性之一，所以除手册之外，还有很多从其他用户获得帮助的方式，如 Blender Chat、Stack Exchange 和 Reddit。

对于工作室和组织，有企业支持，对于希望将 Blender 添加到他们的解决方案的工作室，Blender 工作室里有包含有关此主题的文档和培训材料。如果你认为你发现了 Blender 的问题，请上报错误。

在支持页面可以找到更多相关细节。

开发

由于开源，Blender 大多数开发工作都是由志愿者独自完成的。开发之间的交流多数是通过以下三个平台：

  • projects.blender.org 系统

  • 各种邮件列表

  • 在线聊天（见下文）

如果你对帮助开发 Blender 有兴趣，可以打开参与页面。

Blender Chat

对于在线实时讨论，我们用的是使用 Blender ID 验证的 blender.chat。

你可以加入以下频道：

  • #today 从社区获取解答。

  • #blender-coders 用于开发者讨论 Blender 开发。

  • #python 使用 Python API 的开发者支持。

  • #docs Blender 文档相关讨论。
