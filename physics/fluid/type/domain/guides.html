辅助（引导）

参考

面板:

物理 ‣ 流体 ‣ 引导

类型:

域

流体导向器是用来在模拟上施加力的。它们就像简单的外力，但也力求保持流体的物理精确流动。导向*面板允许你在全局范围内调整导向力，也就是说，在整个领域内。启用引导力会提示流体求解器使用更精确的，但也是计算量更大的压力求解步骤。

即使没有引导对象出炉或者没有连接引导域，如果引导功能被启用，流体求解器仍然会执行更昂贵的压力引导算法。因此，建议只在有明确意图在模拟中使用导引时才启用 导引。

See also

流体导向是 流体的原始-双重优化 的实现。

速度系数

控制导引的滞后性。一个较大的值（也被称为 "α "引导值）会导致更大的滞后。

尺寸

这个设置决定了导引产生的漩涡的大小。更大的引导尺寸（也称为模糊半径或`β'引导值）会产生更大的旋涡。

速度系数

所有导引速度都乘以这个系数。也就是说，导引网格的每个单元，其大小与领域对象相同，都乘以这个系数。

速度源

引导速度可以来自域内运动的物体，也可以来自其他流体域。

效果器

域内的所有效应器对象都将被视为全局引导速度网格。一旦效应器对象被烘焙，就不可能再改变流体领域的分辨率。

域

当使用另一个流体域作为导引速度源时，该域可能有不同的分辨率，也可能是不同的类型（例如导引域是 气体 类型，而其中有导引效果的实际域是 液体 类型）。

为了使用一个域作为速度源，这个域需要已经被烘焙。

引导父级

当使用 域 作为速度源时，该字段用于选择导引域对象。

烘焙指南，免费指南

This option is only available when using the Modular cache type and when using Effector as the Velocity Source. Bake Guides writes vertex velocities of effector objects to drive. It is meant to be used before baking the fluid simulation.

进展将显示在状态栏中。按 Esc 将暂停模拟。

一旦模拟被烘焙，可以按 释放引导 删除缓存。可以暂停或恢复 烘焙引导 进程。
