粒子编辑模式

使用 粒子系统编辑模式 编辑关键点（关键帧）和 路径 烘焙 毛发、粒子、布料 和 软体 模拟器。（烘焙前你可以编辑毛发的造型。）

由于在粒子编辑模式下工作非常简单，并且与在3D视口中处理顶点非常相似，因此我们将展示如何设置粒子系统，然后给出各种功能的参考。

Important

Particle Edit Mode, specifically for hair is deprecated; please use the new 空白毛发 object with its associated Sculpt Mode instead.

用法

Tip

只有烘焙进内存的帧是可编辑的！

如果您无法编辑粒子，请检查您是否正在烘焙 磁盘缓存。

毛发粒子系统的设置

  1. 创建一个 毛发 粒子系统。

  2. 在 法线 方向给一个初始速度。

  3. 创建一个模拟。

  4. 检查 毛发动态 框。

粒子，布料，软体模拟的设置

  1. 使用 毛发 粒子，或布料/软体模拟。

  2. 通过设置对象或发射器来创建模拟，设置您的时间范围（如果您刚刚开始并尝试，请使用小范围），使用 Alt-A 预览来模拟设置。

烘焙模拟

当对模拟满意后，在物体模式开始 烘焙 模拟。模拟必须烘焙后才可编辑。

编辑模拟

Switch to Particle Edit from the Mode select menu in the header of the 3D Viewport to edit the particle's paths/Keyframes. You may need to press T from within the 3D Viewport to see the Particle Edit panel. Move to the frame you want to edit and use the various Particle Edit tools to edit your simulation. Work slowly, previewing your changes with Alt-A, and save often so that you can go back to the previous version should something happen, or that you do not like the latest changes you have made.

Tip

To be able to clearly see what you are working on:

  1. Open the Options panel in the Toolbar.

  2. Select Point select mode (see below) in the header of the 3D Viewport. This will display key points along the particle path.

选择

  • 单选：鼠标左键。

  • 全选: A。

  • Linked: Move the mouse over a keypoint and press L.

  • 框选: B。

  • 圈选 C。

  • 套索选择 Ctrl-Alt-鼠标左键。

  • 根级/末级：选择 ‣ 根级/末级。

You may also use the Select Menu.

Tip

选择

Selections are extremely useful for modifying only the particles that you want. Hover over a particle path and press L to link-select it, hover over the next and press L to add that path to the selection. To remove a path, hold Shift and press L. To Deselect all press A.

The method to select individual points is the same as in Edit Mode. RMB to select, Shift-RMB to add/remove a point from the selection.

随机选择

随机选择粒子。

百分比

按百分比随机选择。

随机种

随机选择的可选种子。

动作

随机选择或者取消选择粒子。

类型

选择毛发或点。这里的这些属于可能会令人困惑，因为毛发/点不是指粒子类型，而是指毛发/粒子的路径/点。

选择模式

路径

没有关键点可见，只有全部粒子可以选择或者取消选择。

点

查看所有关键点。

梢

查看并编辑（包括笔刷）粒子的尖端，即最后一个关键点。

工具

参考

工具栏

梳理

移动关键点（类似于比例编辑工具）。

偏转发射源

仅限毛发粒子--不要通过发射网格移动关键点。

距离

与发射器保持距离。

平滑化

平行视觉相邻的细分。

相加

添加新粒子。

数量

每步的新粒子数。

插值

从现有的毛发中插入新毛发的形状。

步数 (阶梯)

梳子步骤的数量。

快捷键/关键体

制作新粒子的关键帧数量。

长度

缩放部分，使毛发的长度不再 增长 或 变短的 收缩。

增长/收缩

设置画笔的添加效果或反转效果。

蓬松

围绕第一个关键点（根）旋转毛发，使毛发 添加 后立起来或者到 减少 后平压。

蓬松量

应用喷发到未选择的终点，（帮助在膨胀根步时保持毛发量）

剪切

缩放段直到最后一个关键点到达画笔。

权重

这对于软体动画特别有用，因为权重定义了软体的 目标。一个关键点的权重为1将不能移动，权重为0的关键点完全是软体动画。该值按照强度 最小 至 最大 的软体目标范围缩放。

通用选项

在画笔类型下方，会显示起设置：

半径 F

设置画笔半径。

强度 Shift-F

设置画笔效果的强度（不适用于添加画笔）。

切割粒子至形状

外型物体

一个网格对象，其边界由 形状切割 工具使用。

剪切

此修饰工具将毛发修剪为由 形状对象 定义的形状。与使用切割工具相比，这是避免突出毛发部分延长的更快方法。它适用于具有大量毛发的角色，使用切割工具在单个平面中工作变得乏味。

编辑

移动关键点或粒子

  • 要移动选定的关键点，请按 G 或使用其他各种方法之一移动顶点。

  • 要移动粒子根，您必须关闭工具栏中的 保持根。

  • 可以执行许多操作，例如顶点，包括缩放，旋转和移除（完整粒子或单个键）。

  • 可能无法复制或挤出键或粒子，但可以细分添加新关键点的 粒子 ‣ 减去。

  • Alternatively you can re-key a particle Particle ‣ Rekey.

显示毛发和粒子路径的平滑程度取决于 工具栏中的 路径步骤 设置。低设置会在点之间产生块状插值，而高设置会产生平滑的曲线。

镜像

参考

模式:

粒子编辑模式

菜单:

粒子 ‣ 镜像

如果要创建X轴对称修发，则必须执行以下步骤：

  1. 选择所有粒子 A。

  2. 使用 粒子 ‣ 镜像 来镜像粒子。

  3. 在 侧栏区域 ‣ 工具 ‣ 选项 打开 X 镜像。

镜像后两个粒子可能会占据几乎相同的位置。由于这会浪费内存和渲染时间，因此可以使用 粒子 菜单中的 按距离合并。

显示/隐藏

参考

模式:

粒子编辑模式

菜单:

粒子 ‣ 显示/隐藏

隐藏和取消隐藏粒子的工作方式与 3D 视口中的顶点类似。选择要隐藏的粒子的一个或多个关键点，然后按 H 。粒子实际上并没有消失，只有关键点。

隐藏的粒子（即隐藏关键点的粒子）不会对各种画笔产生反应。但：

如果使用 镜像编辑，即使移动了镜像对应物，也可以移动具有隐藏关键点的粒子。

要取消隐藏所有隐藏的粒子 Alt-H。
