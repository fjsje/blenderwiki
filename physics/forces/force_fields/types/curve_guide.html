曲线引导

参考

面板:

物理 ‣ 力场

类型:

曲线引导

所述 曲线引导 用于迫使粒子跟随由定义的某个路径 曲线物体。典型的情况是移动静脉内的红细胞，或使动物中的颗粒流动。您还可以使用 曲线引导 来塑造某些发丝。

Note

您还可以使用 粒子编辑模式 来定义路径。

由于您可以将曲线动画化为软体或任何其他寻常的方式，您可以构建非常复杂的动画，同时保持很好的控制并将模拟时间保持在最低限度。

The option Curve Follow does not work for particles. Instead you have to set Angular Velocity (Particle system tab) to Spin and leave the rotation constant (i.e. do not turn on Dynamic).

曲线导轨 力影响同一层上的所有粒子，与它们到曲线的距离无关。如果图层中有多个指南，则它们的字段会相互加起来（您可能在物理课程中学到的方法）。但是您可以通过更改 最小距离 来限制它们的影响半径（见下文）。

颗粒在其寿命期间遵循 曲线引导，速度取决于其寿命和路径的长度。

Note

曲线引导不影响 软体。

选项

自由

粒子寿命的分数，不用于曲线。

衰减强度

此设置控制 最小距离 和 最大距离 之间的引导强度。衰减为 1 表示线性递增。

添加

如果使用 添加，则还会根据衰减评估粒子的速度。

权重

使用曲线权重来影响沿曲线的粒子影响。

簇集数量

颗粒在曲线（1）的末端聚集在一起，或者它们分开（-1）。

形状

定义颗粒聚集在一起的形式。+0.99：颗粒在曲线末端相遇。0：沿曲线的线性进展。-0.99：曲线开始时颗粒相交。

最小距离

从曲线到力场以全强度有效位置的距离。如果衰减为 0，则此参数将不起作用，因为该场在最大强度达到 最大距离（或无穷大）时有效。最小距离 在 3D 视口中曲线端点处用一个圆显示。

最大距离

最大影响半径。由曲线对象周围的附加圆圈显示。

Warning
