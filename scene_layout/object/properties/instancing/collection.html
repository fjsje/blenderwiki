集合

参考

模式:

物体模式

面板:

属性 ‣ 物体属性 ‣ 实例化 ‣ 集合

实例化集合 用于为每个其它物体的实例创建实例化集合。集合可能包含动画、带物理模拟的物体和其它嵌套的集合。

基础用法

创建一个集合：

  • 创建一个新集合（可在大纲视图中完成）。

  • 关联需要被实例化的物体为新创建的集合的一部分。

创建一个新集合实例：

  • 添加 ‣ 集合实例

此时，将出现集合的实例和空物体。您可以复制空物体，并为每个空物体保留 实例化集合 设置。这样，您可以非常轻松地获得关联数据的多个副本。

集合和动态关联

参见追加与关联以了解如何将来自另一个 blend 文件的数据动态关联到当前文件中。可以将集合从一个 blend 文件动态关联到另一个 blend 文件。执行此操作时，在创建控制集合实例出现位置的物体之前，关联集合不会出现在场景中的任何位置。

创建一个真实的实例集合

如果要对实例化集合进行进一步编辑，请选择 实例集合。然后调用实例独立化将集合转换为可以正常转换和动画的常规对象。

Note

如果实例化集合由外界文件关联而来，物体数据（网格、材质、纹理、变换）将始终从源合集关联而来。但各类物体的父子关系不复存在。
