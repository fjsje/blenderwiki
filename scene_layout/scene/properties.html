场景属性

场景

参考

面板:

属性编辑器 ‣ 场景 ‣ 场景

摄像机

用于选择哪个相机作为活动相机。还可以在3D视图使用 Ctrl-数字键盘0 设置活动相机。

背景场景

可使用一个场景作为背景，当想专注于动画的前景，且不想背景元素碍事时这很有用。

该场景可以有自己的动画、物理模拟等，但如果用户要编辑其中的任何内容，必须从 场景 数据块菜单中选择它。

背景场景本身可以具有背景场景（它们以递归方式包含）。因此，您始终可以通过将现有场景用作新创建的场景的背景来添加内容，从而进行添加。

Tip

这也可以与链接到场景结合使用，一个 blend 文件包含了环境，可以在许多地方重复使用。

活动剪辑

Selects a Movie Clip that can be used by 运动追踪 Constraints or a camera's 背景图.

单位

参考

面板:

属性编辑器 ‣ 场景 ‣ 单位

单位系统

用于用户界面控件的单位系统。

无:

使用与现实世界无关的单位，实际上这与 公制 相同，只是没有单位名称。

公制:

在场景中使用公制单位系统。

英制:

在场景中使用英制单位系统。

缩放单位

在内部单位和用户界面中显示的值之间进行转换时使用的比例系数。当在微观或天文尺度上建模时，可以改变该系数。

Note

这仅影响用户界面中显示的值，而不影响内部的行为方式。例如，物理模拟不考虑单位比例。

分离单位

使用 公制 或 英制 时，显示多个单位。如 2.285m 会显示为 2m 28.5cm 。

旋转

用于显示/编辑旋转值的单位。

角度:

在用户界面以角度为角的计量单位。

弧度:

在用户界面以弧度为角的计量单位。

长度

将用于显示长度值的单位。

自适应:

用于特定值的单位取决于值的大小。例如，某些值可能显示为 23cm，而其他值则显示为 10km 。

米/厘米/英尺:

一个固定的单位，用于用户界面中的所有长度。

质量

见 长度。

时间

见 长度。

温度

见 长度。

音频

参考

面板:

属性编辑器 ‣ 场景 ‣ 音频

用于控制全局音频设置的选项。关于控制声音如何在 Blender 中播放，见偏好设置中的音频设置。

体积/音量

该场景的音量。

间距样式

根据距离更改声音衰减的计算方式。物理上最正确的是 反向 模型，但也可以选择线性和指数衰减。钳制模式将音量限制为低于 100%（1.0），这意味着如果距离小于参考距离，则音量始终为 100% 。有关每个选项的确切描述，参见 OpenAL 文档。

多普勒速度

多普勒效应计算的声音传播速度。典型数值为空气中 343.3 m/s，水中大约 1560 m/s 。

多普勒系数

控制多普勒效应作用强度。你可以放大或缩小音调的变化，但物理真实系数为 1.0 。

更新动画缓存

更新音频动画缓存。这在你意识到音频存在瑕疵时很有用。
