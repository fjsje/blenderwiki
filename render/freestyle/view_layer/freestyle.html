Freestyle

参考

面板:

属性 ‣ 视图层 ‣ Freestyle

每个视图层只有一个视图映射。它控制边缘检测参数。可以通过切换面板标题中的复选框来启用/禁用每个视图层的 Freestyle。

控制模式

用于检测实际上已渲染的边缘，并且将从头到尾地执行控制：

参数编辑器模式:

线条通过用户友好界面中定义的参数进行渲染，以定义和控制线条集和线条样式。

视图地图（因此为视图图层）可以具有多个线集，并且每个线集都链接到一个线型。

Python 脚本编写模式:

线条通过 Python 脚本 进行渲染，强大但复杂。

视图映射缓存

该选项用于重用先前计算的视图映射，以供后续渲染。当输入三维场景的网格几何结构被改变时，缓存将自动更新。

此功能提供了一个主要的性能提升，就是在Freestyle的动画渲染时，当相机空间网格的几何结构是静态的，如同重复，则依然以直线化选项进行渲染。

尽管 视图映射缓存 复选框是一个视图层选项，但缓存内存由所有视图层和场景共享。这意味着，如果将 Freestyle 用于两个或多个视图层（可能通过合成器在不同场景中），则一个视图层的缓存视图将被另一个新视图替换，因此不会预期性能提升。

作为渲染通道

Freestyle 线条不会立即显示在渲染图像的顶部。相反，Freestyle 线条被渲染为 渲染通道，该图像可以与具有 Alpha 上叠节点的渲染图像合成。

边缘检测

折痕角

如果两个相邻的面形成小于所定义的 角阈值 的角，则在使用线集中的边缘类型选择时将呈现它们之间的边缘。该值也影响 轮廓 边缘类型选择。

剔除

忽略不在视野之外的边缘。（节省一些处理时间和内存，但在某些情况下可能会降低结果的质量。）

面平滑度

在计算边缘时考虑 平滑阴影。

球半径

影响线条集中的 脊、谷 和 影射轮廓 边类型选择的曲率计算。每个顶点的曲率是通过在指定半径内对表面的形状求平均值来计算的。增加该值可减少噪点和细节。

极小 Kr 导数

Controls the threshold on the minimum rate of change of curvature used to filter the output of the Suggestive Contour edge type selection. Increasing the value reduces the amount of rendered lines, starting from smoother areas of the object (further information in this pdf).
