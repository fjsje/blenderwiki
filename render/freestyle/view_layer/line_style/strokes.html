笔画

描边是最终渲染的线条。然而，您可以调整它们，例如，通过删除比某些阈值更长/更短的线条，将线条链接成单个笔画或根据角度，虚线图案等将一笔画分成多个笔画。

端盖样式

您可以在三种类型的线帽之间进行选择：

底盖:

平帽，正好在线的终点处。

圆（四舍五入）:

以直线端点为中心的半圆。

正方形:

以直线端点为中心的正方形（因此，与圆一样，与计算值相比，线的绘制端略微延伸）。

链形

默认情况下，从线集中检索到的所有行都链接在一起。有两种基本的链接方法：

方法
朴素:

默认链接方法；它创建简单的链。

素描风格:

此链接选项允许生成具有粗略多笔触的特征边链。基本上，它生成 圆形 笔画而不是单个笔画。只有在线条样式中使用一些随机驱动的修改器时，它才真正有用！

圈数

它以粗略的描边指定倒圆角数。

相同的物体

如果为 true，则仅连接同一对象的要素边。

还可以关闭链接以单独呈现每条线，这对于依赖于线集准确表示的线型非常有用。

拆分

您可以通过启用以下功能之一来拆分 Freestyle 线条链：

最小/最大 2D 角度

当特征边的 2D 角度高于（或低于）最小（或最大）阈值时，拆分特征边链。

2D 长度

在链长于给定值时拆分链。

材质边界

分割特征边的链（如果它们从一个材质交叉到另一个材质）。

拆分样式

使用给定的虚线图案拆分链（另请参见虚线）。

虚线 1、2、3

用于拆分的指定虚线段的长度。

间隙 1、2、3

用于拆分的指定间隙的长度。

排序

您可以对笔画的顺序进行排序，允许线条按给定的顺序堆叠。

排序键

排序键用于确定行的堆叠顺序。

到相机的距离:

靠近相机的线条位于更多线条的顶部。

2D 长度:

较长的线条位于较短的线条之上。

投影 X/Y:

按影像坐标系中的投影 X 或 Y 值排序。

插值类型

与排序键结合使用可确定排序范围。由于线与照相机的距离可能因顶点而异，因此此选项根据在各个顶点处计算的值计算线的排序键。为该行计算的值为：

平均值:

为顶点获取的值的平均值。

最小值:

为顶点获取的值的最小值。

最大值:

为顶点获取的值的最大值。

起始点:

为第一个顶点获取的值。

结束点:

为最后一个顶点获取的值。

排序次序

对于给定的结果，您可以选择 "反转" 排序顺序。

选择

还可以选择仅呈现选定的链。

最小/最大 2D 长度

链条长和/或短于 2D 长度。

链总数

允许选择前 N 条链。

虚线

通过启用 虚线 复选框，您可以指定三对虚线和间隙长度。虚线值定义虚线描边的长度，而间隙值指定两个破折号之间的间隔。

如果指定了零间隙，则即使相应的短划线具有非零值，也会忽略该破折号。

虚线被视为单独的描边，这意味着您可以应用线帽，以及颜色、Alpha 和粗细修改器。

虚线 1、2、3

虚线的指定短划线的长度。

间隙 1、2、3

虚线的指定间隙的长度。
