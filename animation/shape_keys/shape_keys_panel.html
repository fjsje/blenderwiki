形态键面板

参考

编辑器:

属性

模式:

所有模式

面板:

物体数据 ‣ 形态键

形态键面板用于创建形态键。

活动的形态键编号

列表视图。

值/帧（数字）

在相对模式中：值是用于在形态（值= 1.0）和其参考键（值= 0.0）之间进行混合的形态键的当前影响值。参考键通常是基型形态。混合的权重可以在1.0以上和0.0以下推算。

在绝对模式中：值是 估算时刻，形态将在此时产生最大影响。

开关（复选标记）

如果未选中，则在混合形态键堆栈到3D视图中的结果可见时，不会考虑该形态键。

形态键专用项
混合后的新形态

添加一个具有该物体当前形变形态的新形态键。这与列表的 + 按钮不同，因为它总是独立于当前混合形态复制基型形态。

镜像形态键

如果在 物体模式 中，网格是对称的，您可以镜像X轴上的形态键。除非网格顶点完全对称，否则这将不起作用。在 编辑模式 中使用 网格 ‣ 对称 工具。

镜像形态键（拓扑）

同 镜像形态键 类似，但是它根据网格的拓扑检测镜像顶点。这种情况下，网格的顶点不用完全对称也可以。

合并为形态

从不同的物体转移当前结果的形态。

选择要复制的物体，然后选择要复制到的物体。使用此操作，一个新的具有第一个物体的当前混合形态的形态键将被添加到活动物体。

传递形态键

从其他物体转移活动形态键而不管它的当前影响值。

选择要复制的物体，然后选择要复制到的物体。使用此操作，一个新的具有第一个对象的活动形态键的形态键将被添加到的活动物体。

删除所有形态键

删除所有形态键以及它们对网格产生的任何效果。

应用所有形态键

将当前可见形状保存到网格数据并删除所有形态键。

相对

将形态键设置为 相对 或 绝对。请参阅 相对或绝对形态键。

形态键锁定（图钉图标）

在不混合的情况下显示在3D视图中的活动形态。当对象处于 编辑模式 时，形态键锁定 自动启用。

形态键编辑模式（编辑模式图标）

如果启用，则在进入 编辑模式 时，活动形态键将 不会 作为默认值产生最大影响。相反，当前形态键的混合形态将是可见的，并且可以从该状态进行编辑。

添加静置位置

Creates an Attribute in the vertex domain called rest_position which is a copy of the position attribute before shape keys and modifiers are evaluated. Only mesh objects support this option.

相对形态键

请参阅 相对或绝对形态键。

With relative shape keys, the value shown for each shape in the list represents the current weight or influence of that shape in the current Mix.

清除形态键 X

将所有影响值或权重设置为零。快速保证3D视图中显示的结果不受形态影响时有用。

数值

形态键与其参考键（通常为基型形态）之间的混合权重。

值 0.0 表示参考键的 100% 影响和形态键的 1.0 。

范围

活动形态键的影响值的最小和最大范围。当 值 低于0.0或高于1.0时，Blender可以推断结果。

顶点组

将活动形态键形变限制为顶点组。通过将临时顶点组指定给复杂形态并将结果复制到新的更简单形态，可以将复杂形态分解为组件。

相对于

选择要形变的形态键。这称为该形态的 参考键。

Note

Rather than storing offsets directly, internally relative keys are stored as snapshots of the mesh shape. The relative deformation offsets are computed by subtracting Reference Key from that snapshot.

Therefore, replacing the Reference Key has the effect of subtracting the difference between the new and old reference from the relative deformation of the current key.

绝对形态键

请参阅 相对或绝对形态键。

使用绝对形态键时，列表中每个形态显示的值表示该形态键将处于活动状态的 估算时刻。

重置形态键时序（时钟图标）

绝对形态键按列表中的顺序以恒定间隔计时。此按钮可重置形态键的时间。如果删除或重新排序帧，则非常有用。

插值

控制形态键间的插值。

线性、原始、Catmull-Rom、B 样条

估算时刻

控制形态键的影响。拖动以查看当前配置的效果。通常，此属性可以插帧动画或绑定驱动器。
