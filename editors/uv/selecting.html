选择

与 3D 视口一样，UV 编辑器的标题栏中也有选择模式按钮和 选择 菜单。

选区同步

如果关闭（默认设置），UV 编辑器只会显示在 3D 视口中选中的面。在一个编辑器中选择一个项目并不会自动在另一个编辑器中同步选中该项目。如果一个 3D 顶点/边对应多个 UV 顶点/边，可以单独选择每个顶点/边。

如果开启，UV 编辑器将始终显示所有面。在一个编辑器中选择一个项目，也会在另一个编辑器中同步选中它。如果一个 3D 顶点/边对应多个 UV 顶点/边，则不能单独选择这些顶点/边（只能选择所有顶点/边）。

选择模式

顶点:

1 选择顶点。

边:

2 选择边。

面:

3 选择面。

孤岛:

4 选择连续的面组。仅当 同步选择 被禁用时可用。

如果启用了 同步选择，您可以在点击选择模式时按住 Shift 来同时激活多种选择模式，或按住 Ctrl 来扩展/收缩选择。

See also

网格选择

粘性选择模式

自动选择其他 UV 顶点的选项。仅在 同步选择 被禁用时可用。

禁用

每个 UV 顶点都可以独立于其他元素单独选择。

共享位置

自动选择与相同网格顶点相对应且具有相同 UV 坐标的 UV 顶点。这是默认设置，会给人一种 UV 贴图中的多个面可以共享同一个顶点的错觉；而实际上，它们有各自重叠的顶点。

共用顶点

自动选择对应同一网格顶点的 UV 顶点，即使它们的 UV 坐标不同。这也是启用 同步选择 时的行为。

选择菜单

全部 A

选择所有 UV 元素。

无 Alt-A

取消选择所有 UV 元素。

反转 Ctrl-I

反转当前的选择。

框选 B

请参阅框选。

框选已钉固项 Ctrl-B

类似于 框选，但只会选中已钉固的 UV 顶点。

刷选

请参阅刷选。

扩展/缩减选区 Ctrl-数字键盘加+, Ctrl-数字键盘减-

依相邻元素扩展/缩减选中项。

选择已钉固项 Shift-P

选择所有已钉固的 UV。

选择相连元素
相连项 Ctrl-L

选择连接到当前所选内容的所有元素。

最短路径

选择两个选中元素之间的路径。（见下文）

选择相似 Shift-G

选择与活动元素在某些方面相似的 UV 元素。调整上一步操作面板提供了几个选项：

类型

要比较的属性。哪些属性可用取决于选择模式。

顶点选择模式
钉固:

选择具有相同钉固状态的顶点。

边选择模式
长度:

选择在 UV 中具有相似长度的边。

3D长度:

选择在 3D 网格中具有相似长度的边。

钉固:

选择钉固状态相同的边。

面选择模式
面光:

选择在 UV 贴图中具有相似面积的面。

3D面积:

选择在 3D 网格中具有相似面积的面。

材质:

选择具有相同材质的面。

物体:

选择属于同一物体的面。当编辑模式下同时存在多个物体时很有用。

多边形边数:

选择具有相同边数量的面。

翻转/未翻转:

选择具有相同朝向的面（在 UV 中面朝上还是面朝下）。

孤岛选择模式
面光:

选择在 UV 中具有相似面积的孤岛。

3D面积:

选择在 3D 网格中具有相似面积的孤岛。

孤岛中的面数:

选择拥有相同面数的孤岛。

比较

比较操作。

等于:

选择值相等的元素。

大于:

选择值较大或相等的元素。

小于:

选择值较小或相等的元素。

阈值

对几乎相同但不完全相同的值的容差。阈值越高，选择的元素就越多。

拆分选中项 Y

“分离” 选中的面，以便将其移动到其他地方而不影响其邻近面。

Hint

与用于网格的拆分选中项物理断开面的操作不同，这是一个纯粹的选择操作。在 UV 空间中，这些面从一开始就没有连接在一起；只是因为 粘性选择 自动选择了相邻面的顶点，所以看起来才会这样。选择拆分 则会再次取消选择这些顶点。

作为 选择拆分 的替代方法，您可以将 粘性选择模式 设置为 禁用。

选择重叠

选择所有互相重叠的 UV 面。

最短路径

参考

模式:

编辑模式

菜单:

选择‣ 选择相连元素 ‣ 最短路径

快捷键:

Ctrl-鼠标左键

沿两个元素之间的最短路径选择所有 UV 元素：使用菜单激活时选择两个元素，使用快捷键激活时选择活动元素和点击的元素。

捷径跨面

对于顶点：允许路径跨越面，沿着面的对角线而不是边游走。

For edges: selects disconnected edges that are perpendicular to the path (edge ring), rather than connected edges along the path (edge loop).

对于面：允许路径穿过仅共享一个顶点而非一条边的面。

拓扑距离

计算距离时，只需计算边的数量，而不测量边的长度。

填充区域 Shift-Ctrl-鼠标左键

选择所有最短路径（而不是只有一条）。

虚线选项

只允许以固定间隔选择元素，创建 “虚线” 而非连续线。

弃选项

重复次序中跳过的元素数量。

选中项

重复次序中选择的元素数量。

偏移量

要偏移序列的元素个数。

选择循环边

参考

模式:

编辑模式

快捷键:

Alt-鼠标左键，或 Shift-Alt-鼠标左键 以扩展现有选中项。

点击边时按住 Alt 键会选中该边，然后将选中的边尽可能向与之平行的两个方向扩展。（这当然适用于选择网格四周的边 “循环”，但在没有循环的情况下也同样有效。）

您可以额外按住 Shift 键以扩展当前选中项而不是将其替换。

See also

网格编辑中的选择循环边。

选择并排边

参考

模式:

编辑模式

快捷键:

Ctrl-Alt-鼠标左键，或 Shift-Ctrl-Alt-鼠标左键 以扩展现有选中项。

按住 Ctrl-Alt 的同时单击一条边会选中该边，然后将选中的边尽可能地向与之垂直的两个方向扩展。（这当然适用于选择网格四周的边 “环”，但在没有环的情况下也同样有效。）

您可以额外按住 Shift 键以扩展当前选中项而不是将其替换。
