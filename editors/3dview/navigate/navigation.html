导航

环绕轨道

参考

模式:

所有模式

菜单:

视图(View) ‣ 导航(Navigation) ‣ 环绕轨道(Orbit)

快捷键:

鼠标中键、数字键盘2、数字键盘4、数字键盘6、数字键盘8。

通过在视口区域上单击并拖动 鼠标中键，围绕关注点旋转视图。

鼠标右键 取消滚动操作。

Alt 键对环绕有几种影响：

  • 使用 Alt-鼠标中键 单击一个点作为关注点：成为视图环绕的中心点。

  • 按住 Alt，然后用 鼠标中键 向某个方向拖动将对齐视图到轴并使其正交。

  • 使用 鼠标中键 拖动，然后按住 Alt 将执行环绕，同时捕捉到世界轴以及它们之间的对角线。

要更改不连续步骤中的查看角度，请使用 数字键盘8 和 数字键盘2 上下移动，或使用 数字键盘4 和 数字键盘6 左右移动。您还可以按 数字键盘9 切换到视图的另一侧（使摄像机围绕 Z 轴旋转 180°）。

  • 环绕风格设置

  • 自动透视设置

扭转

参考

模式:

所有模式

菜单:

视图 ‣ 导航 ‣ 左倾/右倾

快捷键:

Shift-数字键盘4, Shift-数字键盘6

默认情况下，将视口摄像机围绕其查看方向以 15° 不连续步长旋转。请参阅旋转角度设置进行配置。

要重置滚动，您可以使用 数字键盘3 将视图与全局 X 轴对齐，然后环绕返回到常规透视图。

鼠标右键 取消翻滚操作。

平移

参考

模式:

所有模式

菜单:

视图(View) ‣ 漫游(Navigation) ‣ 平移(Pan)

快捷键:

Shift-鼠标中键、Ctrl-数字键盘2、Ctrl-数字键盘4、Ctrl-数字键盘6、Ctrl-数字键盘8

向上、向下、向左和向右移动视图。要平移视图，请按住 Shift 并在 3D 视口中拖动 鼠标中键。对于不连续操作，可使用热键 Ctrl-数字键盘8、Ctrl-数字键盘2、Ctrl-数字键盘4 和 Ctrl-数字键盘6，就像使用环绕功能一样。

鼠标右键 取消滚动操作。

放大/缩小

参考

模式:

所有模式

菜单:

视图(View) ‣ 导航(Navigation) ‣ 放大(Zoom In)/缩小(Zoom Out)

快捷键:

Ctrl-鼠标中键、滚轮、数字键盘加+、数字键盘减-

将视图移到更接近或更远的位置。您可以通过滚动 滚轮 或使用 Ctrl-鼠标中键 拖动来放大和缩小。要使用不连续步骤缩放，请使用热键 数字键盘加+ 和 数字键盘减-。

Hint

如果您迷失在3D空间中（这并不罕见），框显全部和框显所选可用于显示场景的内容。

鼠标右键 取消缩放操作。

框选放大

参考

模式:

所有模式

菜单:

视图(View) ‣ 漫游(Navigation) ‣ 框选放大(Zoom Region...)

快捷键:

Shift-B

缩放区域 工具允许您通过使用 鼠标左键 拖动来指定矩形区域。然后，视图将放大此区域。

您也可以用 鼠标中键 拖动来放大。

滑动视图

参考

模式:

所有模式

菜单:

视图(View) ‣ 导航(Navigation) ‣ 滑动视图(Dolly View...)

快捷键:

Shift-Ctrl-鼠标中键

在大多数情况下，只需缩放视图即可近距离查看某些内容。但是，缩放只会使您达到焦点，而不会再远了。如果要达到缩放不再起作用的位置，则可通过按住 Shift-Ctrl 并用 鼠标中键 上下拖动来滑移。这将移动焦点（以及与之相关的视图）。

鼠标右键 取消滑动操作。
