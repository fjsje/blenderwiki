视图叠加层

参考

模式:

所有模式

标题栏:

叠加层

Clicking the icon toggles all overlays in the 3D Viewport. The drop-down button displays a popover with more detailed settings, which are described below.

The available options depend on the mode that the 3D Viewport is in.

物体模式

下列选项始终存在，与当前模式无关。部分叠加层可以在视图设置中自定义。

辅助（引导）

栅格

在正交侧视图中显示栅格。

基面

在透视视图中显示地平面。

轴向

显示 X、Y 和/或 Z 轴线。

比例|缩放

栅格/基面各线间距离。

细分

栅格线之间的细分数。

文本信息

在视口左上角显示各种信息。

透视视图

Name of the View Perspective, such as "Top Orthographic" or "User Perspective."

播放帧率(FPS)

Displays the Frames Per Second at which the animation is playing. By default, Blender goes through every single frame, which may result in an FPS that's lower than intended (and the animation playing slower than realtime); the FPS turns red in this case. You can change this behavior in the Playback popover of the Timeline.

物体信息

在括号中显示当前帧，之后是所选的集合和活动物体的名称。如果适用，还会显示当前帧上选定的形态键和（在角括号中）标记。如果物体在当前帧上有关键帧，则物体信息将显示为黄色。

栅格分辨率

When the view is aligned to a world axis (see 视图), the Text Info additionally shows the smallest distance between two parallel grid lines.

统计信息

显示物体和几何体的数量信息。请注意，计数取决于当前选择。例如，选择网格会显示顶点、边和面的数量，而选择灯光则会显示场景中灯光的数量。

  • 所选物体的数量和总数。

  • Geometry -- Displays information about the current scene depending on the mode and object type. This can be the number of vertices, faces, triangles, or bones.

HDRI 预览

显示两个球体，一个高光球体和一个漫射球体，以预览用于世界照明的 HDRI。虽然 HDRI 可以在 材质预览 和 渲染 着色模式中使用，但 HDRI 预览叠加仅在前者中可用。

3D 游标

显示 3D 游标。

标注

显示标注。

物体

附加的

显示不具有几何的物体（比如空物体、摄像机和灯光）。

灯光颜色

根据灯光产生的颜色对灯光物体的轮廓进行着色。

关系线

显示表示父项或约束关系的虚线。

所选物体的轮廓线

显示在所选物体周围的轮廓线。

骨骼

显示骨骼。

运动路径

显示运动路径的叠加层。

原点

显示所选物体的原点。

原点（全部）

显示所有物体的原点。

几何数据

线框

显示网格边。与线框着色类似，但会在现有着色的基础上显示边。数值滑块可调整要显示的边：数值越小，几乎平整的表面上的边越会隐藏，而数值为 1 时则会显示所有边。

不透明度

显示边的不透明度，从 0（不可见）到 1（完全不透明）。

淡化非活动几何体

在物体模式以外的其他模式下，淡出非当前处理的物体。滑块可以控制淡出的程度。

面朝向

法线指向摄像机的面显示为蓝色，法线远离摄像机的面显示为红色。这样就可以快速检查出方向不正确的面：物体的外表面通常应该是全蓝色的。

运动追踪

显示运动追踪的叠加层。

摄像机路径

显示重建后的摄像机路径。

标记名称

显示重建后的轨迹物体名称。

轨道

更改重建轨迹的显示方式：纯轴、箭头等。

尺寸

更改重建轨迹的显示尺寸。

网格编辑模式

以下选项在网格编辑模式下可用。

边

Highlight selected and partially selected edges. Only affects vertex and face selection modes, as edges are always highlighted in edge selection mode.

面

高亮显示选中的面。所有模式均受影响。

中心

在实体着色模式下显示面中心点。（面中心点在线框着色模式下始终可见）

仅影响面选择模式。

折痕

显示因使用表面细分修改器标有折痕的边。

锐利

显示锐边，可用于拆边修改器。

倒角

显示为倒角修改器创建的权重。

缝合边

显示 UV 展开缝合边。

着色

重拓扑

Hide the solid mesh and offset the overlay towards the view. Selection is occluded by inactive geometry, unless X-Ray is enabled

偏移量

Amount to offset edit mesh in front of other geometry.

顶点组权重

在编辑模式下显示权重。

零权重

以黑色显示未被引用和零权重区域。这有助于识别被涂上的权重很低的区域。

无:

以正常方式显示顶点。

活动:

在活动顶点组中无权重的顶点将显示为黑色。

全部:

在任意顶点组中无权重的顶点将显示为黑色。

网格分析

显示网格分析叠加层。

测量

显示所选元素的数值测量值。可以在场景属性中设置单位。

边长

显示所选边边长。

边的夹角

显示所选边相邻面之间的夹角。

面的面积

显示所选面的面积。

面内角

显示所选面的各内角角度。

Tip

变换时会显示连接到所选项的几何图形，例如，允许移动顶点并查看连接的边长。

Note

These values respect the Transform Space in the Sidebar. Use Global if you want the object's scale to be applied to the measurements.

See also

测量工具，用以测量任意距离和角度。

法向

  • 显示顶点法线

  • 在顶点处显示面法线（拆分法线）

  • 显示面法线

尺寸

所选法线显示的尺寸。

恒定屏幕法线尺寸

相对于缩放级别保持法线大小恒定。

开发人员

这些叠加层仅当界面设置中的 开发选项 开启后才可用。

编号

显示所选顶点、边线、面的编号。

Freestyle

这些设置应用于 Freestyle 线条艺术渲染器。

边标记

显示 Freestyle 边标记。

面标记

显示 Freestyle 面标记。

雕刻模式

遮罩

将遮罩显示为物体上的叠加层。该叠加层的不透明度可以调节。

面组

将面组显示为物体上的叠加层。该叠加层的不透明度可以调节。

顶点绘制

镂板遮罩不透明度

无任何作用。（镂板遮罩仅适用于纹理绘制。）

显示线框

以白色显示网格边（与 线框 叠加层不同，后者以黑色显示网格边）。

权重绘制

不透明度

叠加层的不透明度。

零权重

以黑色显示未被引用和零权重区域。这有助于识别被涂上的权重很低的区域。

无

以正常方式显示顶点。

活动

在活动顶点组中无权重的顶点将显示为黑色。

全部

在任意顶点组中无权重的顶点将显示为黑色。

显示权重轮廓线

显示由具有相同插值权重的点形成的轮廓线。

这可视化了权重变化太小，无法从颜色中看到，并且可用于例如在使用平滑工具和笔刷时判断梯度的平滑度和一致性。

显示线框

以白色显示网格边（与 线框 叠加层不同，后者以黑色显示网格边）。

纹理绘制

镂板遮罩不透明度

镂板遮罩叠加层的不透明度。

骨骼

淡化几何体

骨骼显示在顶部，其他几何体显示在背面。可以使用滑块控制不透明度。仅在姿态模式下可用。

骨骼线框不透明度

在 线框 着色模式（或使用透视时的 实体 着色模式）下绘制骨骼时使用的最大不透明度。当需要减少杂乱并将注意力集中在网格而不是骨骼上时，这很有帮助。

蜡笔

洋葱皮

显示当前帧前后关键帧的重影。如果启用了多重帧，则显示所选关键帧的重影。参见洋葱皮章节。

画布

Display a grid over the Grease Pencil drawing plane. The opacity of the grid can be controlled with the slider. When using the Canvas X-Ray option, objects are drawn behind the canvas grid.

淡化非活动层

降低除活动层之外的所有层的不透明度。可通过滑块控制不透明度系数。

淡化非活动物体

使用全彩色图层覆盖活动蜡笔物体以外的所有视图区域，以提高在复杂场景上绘制时的可见性。

淡化蜡笔物体

包括或排除蜡笔物体。

编辑线条

在编辑模式下显示编辑线。

仅在多帧中

启用多重帧功能后，如果选择了当前帧以外的关键帧，则这些关键帧上的笔画将仅显示为编辑线，而笔画本身将被隐藏。请注意，这不会影响洋葱皮。

笔画方向

切换显示所选笔画的起始点（绿色）和结束点（红色），以显示其方向。

材质名称

在所选笔画旁显示材质名称。

顶点不透明度

编辑模式与雕刻模式下，顶点/点和编辑线的不透明度。

顶点绘制不透明度

顶点绘制模式和绘制模式下顶点颜色叠加层的不透明度。请注意，在绘制模式下，顶点绘制默认只在 材质预览 和 渲染 着色模式下可见。要在 实体 模式下看到顶点色，需要使用顶点绘制模式，或者将颜色着色设置设置为 属性。

控制柄

When Curve Editing is active, this option controls how curves are displayed in the 3D Viewport.

无:

No handles are displayed, only the control points.

选中项:

Only handles for selected control points are displayed.

全部:

所有控制柄都会显示。
