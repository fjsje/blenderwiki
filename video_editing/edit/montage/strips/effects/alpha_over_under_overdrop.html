Alpha在上、在下 & 向下覆盖

Using the alpha (transparency channel), this effect composites a result based on transparent areas of the dominant image. If you use a Scene strip, the areas of the image where there is not anything solid are transparent; they have an alpha value of 0. If you use a Movie strip, that movie has an alpha value of 1 (completely opaque).

因此，你可以使用 Alpha在上 / Alpha在下 效果将CGI场景合成到影片上。结果为你的模型就如它是电影的一部分一样。Adjust ‣ Compositing ‣ Opacity 控制前景在背景中的混合程度，在背景上的前景淡化。透明前景图像区域的颜色被忽略，不会改变背景的颜色。

Alpha 上叠

使用 Alpha在上 时，片段按所选顺序分层；所选的第一个片段是背景，第二个片段则在选择的第一个片段之 上。不透明度 控制 前景 的透明度，即0.0的 不透明度 将仅会显示背景，而1.0的 不透明度 会使用前景完全覆盖背景（当然除了这个是透明区域的情况！）

Warning

通过单击前景片段侧栏中的预乘Alpha按钮，两个片段的alpha值不会相乘或相加。在具有平滑不透明（alpha=1.0或更大）通道的片段上添加具有可变alpha通道的前景片段（某些不透明区域，某些透明区域，某些介于两种状态间的区域）时使用此效果。当使用 alpha向上 时，如果发现前景物体周围有微光或前景物体的奇怪透明区域，请启用 预乘。

例子

Alpha 在下

使用 Alpha在下 时，这是相反的：所选的第一个片段是前景，第二个片段是背景。此外，不透明度 控制 背景 的透明度，即 不透明度 为0.0；将仅显示前景（背景是完全透明的），1.0的 不透明度 的效果与 Alpha在上 一样。

向下覆盖

向下覆盖 效果是介于另外两个之间：与 Alpha在下 一样，所选的第一个片段将成为前景，但与 Alpha在上 一样，不透明度 控制此前景的透明度。

向下覆盖 效果与过渡效果十分相似，但将设置为顶部或第二个图像，给予更多的渐进覆盖效果，而非像过渡那样的混合效果。当然，所有的Alpha效果都遵从alpha（透明度）通道，而过渡则不是。

施加的 Alpha 程度，以及因此而混合的颜色，可以通过函数曲线进行控制。创建正弦波可能会产生前景淡入淡出的效果。
