自由线工具

参考

模式:

绘制模式

工具:

工具栏 ‣ 绘制

The Draw tool allows you to draw free-hand strokes.

笔刷设置

材质

Data-block selector for the material.

半径

笔刷的半径，单位为像素。

F 允许你通过拖动指针或输入数字然后确认来交互式地改变画笔大小。

Use Pressure (pressure sensitivity icon)

Uses stylus pressure to control how strong the effect is. The gradient of the pressure can be customized using the curve widget.

强度

控制笔触的透明度（alpha）。从完全透明（0.0）到完全不透明（1.0）。

你可以通过在 3D 视口中按 Shift-F，然后移动指针，再按 鼠标左键，来交互地改变画笔的强度。你也可以用数字来输入尺寸。

Use Pressure (pressure sensitivity icon)

Uses stylus pressure to control how strong the effect is. The gradient of the pressure can be customized using the curve widget.

盖类型

笔画的起点和终点的形状。

圆（四舍五入）:

笔画的开始和结束都有一个弯曲的形状。

平展:

笔画以直截了当的方式开始和停止。

高级

输入采样

控制输入设备读取并在笔画上生成点的频率。数值越高，精度越高（更多点），但是产生不规则的笔画；数值越低，精度越低（更少点），但是产生柔和的笔画。（0禁用额外的输入设备采样。）

你必须根据你的输入设备设置合适的值，来平衡笔画的精确度和柔和度。详见 输入设备。

激活光滑

绘制笔画时的平滑迭代次数。

角度

笔画厚度最大时，输入设备的笔画方向（0°为水平方向）。

系数

当笔画垂直于 角度 值时，厚度的减少量。

硬度

边界点到中心的透明（alpha）程度。仅在笔刷使用 点 或 方块 类型的笔画材质时起作用。

宽高比 X/Y

控制alpha渐变的宽、高。

笔画

后期处理

后期处理在完成绘制时执行，即松开 鼠标左键 或 Pen 时。你可以使用面板区域顶栏的复选框切换是否使用后期处理。

平滑化

对笔画上点位置的平滑力度。

迭代

应用于笔画的平滑迭代次数。

细分步数

应用于新创建笔画的细分次数。

简化

使用自适应算法减少笔画中的最最终点数。

修剪笔画末端

自动修剪笔画末端。

轮廓

Activate the conversion of the newly created stroke to its outline.

材质

用于轮廓笔画的材质。

厚（宽）度

用于轮廓笔画的宽度值。

随机

沿着笔画给点的位置添加随机性。你可以使用部分面板标题中的复选框来切换随机化的使用。

半径

使用输入设备的压感用于随机的程度。

强度

应用于笔画强度值（alpha）的随机程度。

UV

应用于 UV 旋转的随机程度。

色相，饱和度，明度

随机化笔画的色相、饱和度和数值，颜色。

抖动

添加到笔画的抖动程度。

通用选项

笔画随机（笔画图标）

只在笔画层面使用随机性。

Use Pressure (pressure sensitivity icon)

使用手写笔的压力来控制效果的强度。压力的梯度可以用 曲线部件 来定制。

笔画防抖

稳定笔画 通过延迟和修正点的位置来帮助减少绘图时笔画的抖动。你可以使用部分面板标题中的复选框来切换 稳定笔触 的使用。

半径

笔锋继续前，到最后一点的最小距离。

系数

一个平滑的因子，较高的值会产生较平滑的笔划，但绘图感就像您在拉动笔锋一样。

游标

可以通过切换 游标 标题中的复选框来禁用游标。

绘图时显示填充颜色

在视图中显示笔刷链接的材质颜色。

用法

选择笔刷和材质

在工具设置中，选择笔刷、材质和颜色类型来使用该工具。绘图工具使用 绘制笔刷 类型。更多信息请参阅笔刷设置。

自由绘制

点按 鼠标左键 或 Pen 在视角绘制自由线。

自由绘制的笔画。

笔画防抖

Shift-鼠标左键 切换使用稳定笔触对画笔的使用，以便在绘图时有更多的控制，获得更平滑的线条。

使用 稳定笔画 绘制笔画。

直线

Alt-鼠标左键 限制画出水平或竖直的线条。

切换擦除工具

Ctrl-LMB changes temporally to the active Erase tool. See Erase Tool for more information.

你可以使用 B 框选，删除选中区域的所有点。
