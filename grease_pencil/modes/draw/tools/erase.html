擦除工具

参考

模式:

绘制模式

工具:

工具栏 ‣ 擦除

The Erase tool erase already drawn strokes.

笔刷设置

半径

笔刷的半径，单位为像素。

F 允许你通过拖动指针或输入数字然后确认来交互式地改变画笔大小。

Use Pressure (pressure sensitivity icon)

Uses stylus pressure to control how strong the effect is.

Occlude Eraser (overlapping squares icon)

Erase only strokes visible and not occluded by geometry.

默认橡皮擦

Use this brush when enabling the eraser tool with the fast switch key (Ctrl).

模式

确定擦除工具的作用方式。

消融:

模拟光栅类的橡皮擦，此类橡皮擦在真正删除点之前会影响笔画的强度和厚度。

点:

一次删除一个点。

笔画:

删除整个笔画。

强度

控制橡皮擦对描边透明度（alpha）的影响程度。

你可以通过在 3D 视口中按 Shift-F，然后移动指针，再按 鼠标左键，来交互地改变画笔的强度。你也可以用数字来输入尺寸。

Use Pressure (pressure sensitivity icon)

Uses stylus pressure to control how strong the effect is.

影响笔画强度

The amount of deletion of the stroke strength (alpha) while erasing.

影响笔画宽度

The amount of deletion of the stroke thickness while erasing.

游标

可以通过切换 游标 标题中的复选框来禁用游标。

用法

选择笔刷

在 "工具设置" 中，选择要与工具一起使用的画笔。"擦除" 工具使用 擦除画笔 类型（柔和、尖点和描边）。

消融擦除

  • 选择软/硬类型的擦除笔刷。

  • 调整笔刷设置。

  • 点按 鼠标左键 或 Pen 删除视角中的笔画。

点擦除

  • 选择点类型擦除笔刷。

  • 调整笔刷设置。

  • 点按 鼠标左键 或 Pen 删除视角中的笔画。

笔画擦除

  • 选择笔画类型擦除笔刷。

  • 调整笔刷设置。

  • 点按 鼠标左键 或 Pen 删除视角中的笔画。
