动画工具

插入空白关键帧

参考

模式:

绘制模式，编辑模式，雕刻模式

菜单:

笔画 ‣ 动画 ‣ 在活动层插入空白关键帧 笔画 ‣ 动画 ‣ 在全部层插入空白关键帧

快捷键:

Shift-I

活动层

在当前帧的活动图层中添加新的空白关键帧。如果当前帧上已有关键帧，则将在下一帧上添加新的空白关键帧。

所有层

启用后，将在所有图层（而不仅仅是活动图层）上创建空白关键帧。

复制活动关键帧

参考

模式:

绘制模式，编辑模式，雕刻模式

菜单:

笔画 ‣ 动画 ‣ 复制活动层的活动关键帧 笔画 ‣ 动画 ‣ 复制全部层的活动关键帧

通过将最后一个关键帧复制到当前帧来进行笔画的复制。

模式
活动

只复制活动层。

全部

复制全部层。

删除活动关键帧

参考

模式:

绘制模式，编辑模式，雕刻模式

菜单:

笔画 ‣ 动画 ‣ 删除活动层的活动关键帧 笔画 ‣ 动画 ‣ 删除全部层的活动关键帧

快捷键:

Shift-X

删除摄影表或当前关键帧中的最后一个关键帧（如果位于其中）。

插值顺序

参考

模式:

绘制模式，编辑模式

菜单:

蜡笔 ‣ 插值顺序

快捷键:

Shift-Ctrl-E

通过添加 多个 关键帧，对上一个关键帧和下一个关键帧之间的笔画插值。将在上一个关键帧和下一个关键帧之间的每个帧上添加一个间断帧。

烘焙网格至蜡笔

参考

编辑器:

3D 视图

模式:

物体和姿态模式

菜单:

物体 ‣ 动画 ‣ 烘焙网格至蜡笔...

将选定帧范围内的网格动画的每个帧转换为带有蜡笔对象的关键帧笔画。烘焙操作 运算符使用应用的所有这些修改器、驱动程序和约束计算所选对象的最终动画，并关键帧结果。

目标物体

Select the target Grease Pencil object for the baked animation or a new one if there is nothing yet.

开始帧，结束帧

烘焙过程的起始/结束帧。

步长

烘焙过程的帧步数

厚（宽）度

笔画宽度。

阈值角度

决定笔画端点的阈值。

笔画偏移

设置偏移以从填充笔画中分离笔画。

仅缝合边

仅转换标记为缝合边的边。

导出面

将面转换为填充的笔画。

仅选定的关键帧

仅转换选中的关键帧。

目标帧

烘焙动画的目标目标帧。

投影类型

设置要用于转换后的描边的投影类型。

开始帧，结束帧

步长

烘焙过程的帧步长。

仅选定的关键帧

仅转换选中的关键帧。

目标帧

烘焙动画的目标目标帧。

投影类型

设置要用于转换后的描边的投影类型。
