  • 蜡笔
  • 简介

简介

蜡笔是Blender对象。它接受来自鼠标或压敏触笔的绘图信息，并将其作为点集合放置在3D空间中，这些点定义为笔画。

The Grease Pencil object can be used to make traditional 2D animation, cut-out animation, motion graphics or used it as storyboard tool among other things.

笔画在 绘制模式 中创建，这需要在蜡笔对象的动画时间轴中新建 关键帧。然后可以在 编辑模式 和 雕刻模式 中调整现有笔画。最后，艺术家可以将材质、修改器、光线和视觉效果应用于笔画。

快速入门

艺术家可以将蜡笔添加到任何现有的Blender场景，也可以从2D动画模板开始。模板提供了一些预配置的选项，有助于动画和故事板。

创建并使用蜡笔

  1. 从 物体模式，添加 ‣ 蜡笔 ‣ 空白。

  2. 创建新的关键帧或打开自动键。（参见关键帧编辑）

  3. 切换到 绘制模式。

  4. 在视口上单击并拖动以向蜡笔对象添加笔触。

二维动画模板

要使用 "二维动画" 项目模板创建新的 Blender 文档，请使用：文件 ‣ 新建 ‣ 二维动画。

请注意以下二维动画模板的预配置设置：

  • 2D 动画是默认的活动工作区。

  • 世界属性 ‣ 表面（背景）‣颜色 设置为白色。

  • 渲染属性 ‣ 颜色管理 设置为标准。

  • 绘画面板 被设置为Front（X-Z）。

  • 线和填充层以及一些笔画材质为蜡笔配置。

  • 当在空帧上使用蜡笔时，动画时间轴将自动创建新的关键帧。

Tip

蜡笔可以从 图形输入板 或手写笔中读取压力敏感度信息。
