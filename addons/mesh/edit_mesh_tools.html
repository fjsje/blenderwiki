编辑网格工具（Edit Mesh Tools）

网格编辑工具为 Blender 增加了几个内置工具中没有的工具，或者为类似任务提供不同的方法。下列每个菜单：顶点、边、面和实用工具是在 侧栏 ‣ 编辑标签 的一个子面板，默认是关闭的。关闭的面板标题栏中的图标包含了一些新的/不同的选择工具和编辑模式顶点/边/面的选择模式，用于在不打开子面板时快速访问一些工具。

在用户界面中，每个类别中都有各种工具，大多数工具都有一个工具提示，提供一些使用信息，帮助用户了解每个工具的基本要求。最重要的快捷键是在编辑模式下双击 鼠标右键 点击后会根据顶点/边/面的选择模式调出顶点/边/面的菜单。菜单集成是将主要工具集成到编辑模式的上下文菜单中，选择工具集成到编辑模式的选择菜单中。

  • Face Inset Fillet：完全基于 zmj100 的插件

  • Vertex Align：作者 zmj100

  • Edge Fillet Plus：作者 Gert De Roost -- 原作者 zmj100

  • Split Solidify：作者 zmj100，由 zeffii 更新为 BMesh

  • Pen Tool：作者 zmj100

  • Mesh Cut Faces：作者 Stanislav Blinov (Stan Pancakes)

  • Vertex/Edge/Face Context Menu：作者 Stanislav Blinov (Stan Pancakes)

  • Edge Roundifier：作者 Piotr Komisarczyk (Komi3D)、PKHG

  • PKHG Face Extrude：由 PKHG 提供，基于 Geodesic Domes 插件的 “面功能”

  • Set Edges Length：作者 "Giuseppe De Marco [BlenderLab] 灵感来自 NirenYang"

  • Edge Tools：作者 Paul Marshall (brikbot)

  • Extrude and Reshape：作者 Germano Cavalcante (mano-wii)

  • Fast Loop：作者 Andy Davies (metalliandy)

  • Multi Extrude：作者 Liero、Jimmy Hazevoet

  • Offset Edges：作者 Hidesato Ikeya

  • Mesh to Wal：作者 luxuy_BlenderCN

  • Vertex Chamfer：作者 Andrew Hale (TrumanBlending)

  • Random Vertices：作者 Oscurart

激活

  • 打开 Blender 并转到 “偏好设置”，然后转到 “插件” 选项卡。

  • 选择 “网格” 然后选择 “Edit Mesh Tools” 以启用插件。

挤出和重塑

You can select a face and extrude it inwards or outwards, creating new faces to accommodate the extrusions. Note: Also known as Push/Pull Face.

松弛

这种操作在尽可能保留形状的同时放松了所选顶点。

要使用这个操作，进入编辑模式，选择你想放松的顶点。要执行这个脚本，可以使用 上下文菜单 ‣ 松弛 或者使用 网络‣ 顶点 ‣ 松弛 来代替。
