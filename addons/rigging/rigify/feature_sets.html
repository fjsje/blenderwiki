功能集

Rigify 允许第三方开发者实现子插件，称之为 功能集，这可以提供新的 元骨骼 和 绑定类型。与常规插件类似，它们可以通过 Rigify 设置从 zip 文件安装。

这些是过去和现在的 Rigify 开发人员目前提供的 功能集 的一些示例：

Cessen 的 Rigify 扩展

此功能集提供了 Nathan Vegdahl 的原始 Rigify 绑定，最低限度地移植和重新打包以在不将 Rigify 切换到旧版模式的情况下工作。请注意，它们的名称已更改，因此为旧版模式设计的元骨骼不直接兼容。

Alexander Gavrilov 的实验性绑定

绑定实验，其中一些将来可能会包含在 Rigify 中。范例包括 膝盖 / 肘部 带有额外 IK 系统的四肢，基于样条线的触手等等。

Cloud Rig

This major conversion feature set by Demeter Dzadik implements the rig used by the Blender Animation Studio using the Rigify framework.

You can install these packages by clicking Clone ‣ Download ZIP, and then install the downloaded file through Rigify settings.

开发者文档可在 Blender Wiki 上获取。
