添加额外的网格物体（Extra Objects）

这个插件把许多创建网格物体的插件组合成一个。

激活

  • 打开 Blender 并转到 “偏好设置”，然后转到 “插件” 选项卡。

  • 选择 “添加网格” 类别，然后勾选 “Extra Objects” 以启用插件。

界面

位于 3D 视图 ‣ 添加 ‣ 网格菜单。

信息

包含的网格物体插件：

添加空物体作为父级 (Parent to Empty)，作者 Liero

Parent the selected mesh to an empty object type.

3D函数曲面 (3D Function Surface)，作者 Buerbaum Martin (Pontiac), Elod Csirmaz

用 XYZ 的数学函数创建物体。包含了 elfnor 创作的几个预设。

横梁建造器 (Beam Builder)，作者 revolt_randy, Jambay

可以创建五种横梁。

齿轮 (Gears)，作者 Michel J. Anders (varkenvarken)

创建齿轮和蜗杆状的网格。

宝石 (Gemstones)，作者 Pontiac, Fourmadmen, Dreampainter, Dominic Kroper, (dommetysk)

创建三种类型的钻石。

蜂巢 (Honeycomb)，作者 Kayo Phoenix

创建蜂窝状图案的网格。

门格尔海绵 (Menger Sponge)，作者 sugiany

创建迭代的门格尔海绵。

管接头 (Pipe Joints)，作者 Buerbaum Martin (Pontiac)

创建弯角或交叉的管道。

阶梯金字塔 (Step Pyramid)，作者 Phil Cote (cotejrp1)

用各种参数创建一个阶梯金字塔。

球形化方块 (Round Cube)，作者 Alain Ducharme (phymec)

创建四边面球体，自带几个预设。

正多面体 (Regular Solid)，作者 DreamPainter

创建多面体物体。

星形 (Star)，作者 Fourmadmen

创建一个简单星形。

超级圆环 (Supertoroid)，作者 DreamPainter

Create a torus object with new parameters for boxed shape and more.

茶壶 (Teapot)，作者 Anthony D'Agostino

传统风格的茶壶和一个中等的勺子。

圆环结 (Torus Knot)，作者 Anthony D'Agostino

一个简单的圆环结，有三种类型。

三角形 (Triangles)，作者 Sjaak-de-Draak

创建基于数学的三角形。

扭曲圆环 (Twisted Torus)，作者 Paulo_Gomes

标准圆环外加一个可以扭曲网格的变量。

添加顶点 (Add Vertex)，作者 meta-androcto, Pablo Vazquez, Liero, Richard Wilks

添加一个单个顶点或物体原点，或以快速开始网格编辑。

墙壁工厂 (Wall Factory)，作者 dudecon, jambay

添加城堡风格的墙壁，附带许多开口设置。
